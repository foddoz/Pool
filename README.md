User Manual: 
CodeBlocks source files located in Pool folder. Executable is located in Executable folder.

Controls:  
Horizontal mouse movement - Change direction of cue stick
Vertical mouse movement - Change power of cue stick
A, D - Rotate camera 1 & 3 around table
W, S - Zoom camera 1 & 3
R - Reset game with rectangular formation of balls
T - Reset game with triangular formation of balls
1 - Top down camera view
2 - Follow cue ball view
3 - 3D perspective view
Esc - Open/close console

Console:
Console opens in a separate window from the game window. Simply press Esc to open console and start typing.
Do not click into the console window, click in the game window instead to type in the console window.
Type "help" and enter to view the list of commands available.
